#### 功能

 * 快捷添加会议室。 如果会议室已被占用，会议室将不会出现在会议室列表中。

 * 加入会议室参加会议。

 * 通过主页清楚地了解会议室概述，包括开始，结束时间，会议摘要等。
 
 * 快速查看收藏的会议室集合。
